description: הדגמת תרחישי שימוש ב-JobSwipe — משתמשי דמו, התאמה, צ'אט
---

הצג לי איך לבחון את התוצר בעצמי:

1. ודא ש-dev server רץ על `http://localhost:3000`. אם לא — הפעל.
2. ודא שיש נתוני דמו. אם קובץ `data/jobswipe.db` לא קיים או נראה ריק, הרץ `npm run seed`.
3. הצג טבלת משתמשי דמו (אימייל, סיסמה `demo1234`, תפקיד, מה כבר עשו) — מצא אותם ב-`scripts/seed.ts`.
4. תן 3 תרחישי בדיקה ממוספרים:
   - **Match מיידי**: כניסה כ-`pizzaria@demo.com` → משרת מלצר/ית → Noa ממתינה → לייק → דיאלוג התאמה.
   - **סוויפ טרי**: כניסה כ-`maya@demo.com` → גרור כרטיסי משרה.
   - **דיווח וחסימה**: כניסה כמועמד שיש לו התאמה, ונסה כפתור "דיווח" או "חסימה" בראש הצ'אט.
5. לכל תרחיש — URL מדויק שאפשר להכניס לדפדפן.

טען את `jobswipe-ops` ו-`jobswipe-orientation`.
